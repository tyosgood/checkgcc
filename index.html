Office 365 GCC Check

Enter a domain below to check if that customer is using Microsoft's GCC, GCC-High or DoD Office 365 cloud services.

About this page

This page uses the fetch() method in javascript on the client to query Microsoft about the status of the entered domain. This query returns the GCC status in JSON, but it's a bit cryptic, so this page interprets the results and returns an easy to understand status. For more information about Microsoft GCC see: Office 365 Government
